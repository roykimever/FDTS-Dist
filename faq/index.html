    📌 자주 묻는 질문¶

    실행 · 로그인¶

    실행하면 관리자 권한 창(UAC)이 떠요.

    정상입니다. 프로그램이 HTS 화면을 자동으로 조작하려면 관리자 권한이 필요합니다. [예] 를 눌러 주세요.

    인증서 로그인에서 멈춰요.
    • 사용할 인증서가 HTS에 정상 등록되어 있는지 확인하세요.
    • [설정] → [계정 설정] 의 로그인 비밀번호가 맞는지 확인하세요.
    • 인증서 창이 뜨는 데 시간이 걸리면, 속도 배수를 올려 여유를 주세요.

    데이터 · 매매¶

    모든 계정에 똑같은 잔고가 들어와요.

    계좌 전환이 제대로 안 된 경우입니다. [설정] → [계정 설정] 에서 각 계정의 계좌 순서(1~N) 가 HTS 계좌 목록의 실제 순서와 맞는지 확인하세요. 최신 버전은 화면 배율에 따른 위치 오차를 자동 보정합니다.

    우클릭/데이터 추출이 안 되거나 오류가 나요.

    화면 해상도·배율 문제일 가능성이 큽니다.

    • [설정] → [일반 설정] → 현재 디스플레이 를 확인하세요.
    • 권장은 3840×2160 · 200% 배율 입니다.
    • 최신 버전은 다른 배율도 자동 보정하지만, 문제가 계속되면 권장값으로 맞춰 보세요.
    레버리지 ETP(SOXL 등) 안내창이 자꾸 떠요.

    프로그램이 자동으로 닫도록 되어 있습니다. 계속 방해가 되면 HTS에서 해당 안내를 미리 확인/해제해 두세요.

    느린 PC에서 자꾸 실패해요.

    [설정] → [일반 설정] → 속도 배수 를 1.5 ~ 2.0으로 올려 대기 시간에 여유를 주세요.

    시트 · 연동¶

    시트를 못 읽는대요 / 권한 오류가 나요.

    내 구글 시트의 [공유] 에 시트 공유 계정(서비스 계정 이메일)을 편집자로 추가했는지 확인하세요. (준비하기 참고)

    텔레그램 알림이 안 와요 / SSL 오류가 나요.
    • [설정] → [일반 설정] 의 Telegram ID/Token을 확인하세요.
    • 회사망 등 보안 프로그램(SSL 프록시) 환경에서는 텔레그램 전송이 막힐 수 있습니다.

    회원 · 승인¶

    ‘미가입’ 또는 ‘승인 대기 중’ 이라고 나와요.

    회원(라이선스) 문서를 참고하세요. 회원가입 후 관리자 승인을 받아야 매매를 실행할 수 있습니다.

    해결되지 않으면 문의하기로 연락 주세요.
