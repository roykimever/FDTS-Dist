    🗒️ 릴리즈 버전 이력¶

    버전별 주요 변경 사항입니다. (최신 버전이 위)

    내 버전 확인

    메인 화면 사이드바 하단의 버전 버튼에서 현재 버전과 최신 여부를 볼 수 있습니다. "…업데이트"로 표시되면 눌러서 최신 버전으로 올려 주세요. (업데이트 방법)

    v6.0.1¶

    • 구글시트 사본 만들기가 한 번에 — [📄 신규 구글시트 만들기] 를 누르면 이제 곧바로 "사본 만들기" 창이 떠서, 클릭 한 번으로 내 드라이브에 사본이 생깁니다. (전에는 템플릿이 열려서 [파일]→[사본 만들기]를 직접 눌러야 했고, 이를 놓치면 여러 사람이 같은 원본 시트를 쓰게 될 위험이 있었습니다.) (준비하기)

    v6.0.0 🎉¶

    • 정식 버전 승격 — 여러 PC에서 실제 매매까지 검증을 마쳐 정식 v6.0.0이 되었습니다. 기능 변경은 없고, 아래 5.9.15~5.9.19의 안정화 작업이 모두 반영된 버전입니다.

    v5.9.19¶

    • 주문 재전송이 안 되던 문제 해결 — "오늘 이미 전송된 주문" 기록이 눈에 안 보이는 폴더에 저장돼, log 폴더를 비워도 계속 건너뛰던 문제를 고쳤습니다. 이제 기록이 log 폴더에 정상 저장됩니다.

    v5.9.18¶

    • HTS 화면 오류 해결 — 매매 중 "DLL을 로드할 수 없습니다" 오류가 뜨던 문제를 고쳤습니다. 이제 HTS를 안전한 방식으로 실행합니다.
    • 앱을 닫아도 프로그램이 남던 문제 해결 — 종료 시 관련 프로세스가 모두 정리됩니다. (폴더가 삭제되지 않거나 업데이트가 실패하던 원인)

    v5.9.16¶

    • 주문 중 갑자기 멈추던 오류 해결 — 로그를 기록하다 프로그램이 중단되던 문제를 고쳤습니다.
    • 로그 글자가 깨지던 문제 해결 — 한글이 정상적으로 표시됩니다.

    v5.9.15¶

    • 설정 저장 방식 개선 — 계정을 여러 개 입력할 때 드롭다운을 옮기면 앞서 입력한 내용이 사라지던 문제를 고쳤습니다. 이제 [💾 모두 저장] 한 번으로 일반 설정과 모든 계정이 함께 저장되고, 저장해도 창이 닫히지 않습니다.

    v5.9.14¶

    • 시트에서 계정 자동 등록 — 구글 시트 링크의 각 탭을 순서대로 읽어 계정을 한 번에 구성. 탭 순서=계좌 순서, 각 탭 D3 셀의 글자를 매매법 이름으로 사용, 매매 탭 수에 맞춰 계정 개수 자동 조절. 등록 전 미리보기 확인창으로 확인. (자세히)

    v5.9.13¶

    • 인증 방식 개선(보안 강화) — 시트 접속에 쓰는 인증 파일을 평문으로 두지 않도록 바꿔, 배포본의 키 노출 위험을 낮춤. 사용에는 변화 없음.

    v5.9.12¶

    • 사이드바 개편 — 버전 버튼 추가(현재 버전과 "최신/업데이트" 상태를 한눈에, 클릭하면 업데이트).
    • About 화면의 블로그 주소를 클릭 가능한 링크로 변경.

    v5.9.11¶

    • 계정을 1개만 운용해도 그 계정만 설정하면 되도록 개선(안 쓰는 계정은 요구하지 않음).

    v5.9.10¶

    • 반자동 자동 업데이트 — 앱 안에서 새 코드를 자동으로 내려받아 무결성 검사 후 교체·재시작. 설정·인증 정보는 그대로 보존.

    v5.9.9¶

    • 새 버전이 나오면 팝업 + 텔레그램으로 알림(자동 실행 중에는 텔레그램만).

    v5.9.8¶

    • 필수 설정(시트·텔레그램·계정)이 미완료면 매매를 막고, 저장할 때 빠진 항목을 안내.
    • 회원 시트에 앱 버전·접속 시각이 기록되도록 함.

    v5.9.7¶

    • 온라인 사용 설명서와 버전 확인을 연결.

    v5.9.5¶

    • 버전 관리 기능 — 구버전 강제 업데이트/사용 제한.

    v5.9.4¶

    • 다양한 화면 배율(100% · 150% · 200%)에서 좌표를 자동 보정해 안정적으로 동작.

    v5.9.3¶

    • 여러 계좌를 순차 매매할 때 계좌별 데이터가 정확히 들어오도록 수정.

    v5.9.2¶

    • 회원가입 화면 정리 + 디스플레이(해상도/배율) 안내 추가.

    v5.9.0¶

    • 회원 인증(관리자 승인) 도입 — 승인된 사용자만 매매 실행.

    문의는 문의하기로 연락 주세요.
