# 💬 문의하기

사용 중 궁금한 점이나 문제가 있으면, **프로그램을 전달해 준 분(제작자)에게 직접 문의**해 주세요.

!!! tip "문의 전에"
    [자주 묻는 질문](faq.md)에서 비슷한 사례를 먼저 확인하면 더 빠르게 해결할 수 있습니다.

!!! note "문의 시 알려주시면 좋아요"
    - 어떤 상황에서 문제가 생겼는지 (예: 로그인 중, 주문 중)
    - 화면 **해상도·배율** (설정 → 일반 설정 → 현재 디스플레이)
    - 프로그램 **로그** 내용 (오류 메시지)
    - 가능하면 화면 **캡처**

---

이용해 주셔서 감사합니다. 🙏
