# 👋 소개

**FDTS AutoTrading**은 메리츠증권 HTS를 자동으로 조작해 미국 주식을 매매해 주는 개인용 자동매매 프로그램입니다. 정해진 매매법(시트)에 따라 로그인 → 잔고·시세 수집 → 주문 → 종료까지의 과정을 사람 대신 수행합니다.

!!! note "이 프로그램은 지인에게만 배포됩니다"
    아무나 받아서 쓰는 공개 프로그램이 아니라, 제작자(로이킴에버)가 아는 분에게만 나눠 드리는 프로그램입니다. 실행하려면 **회원가입 후 관리자 승인**이 필요합니다. 자세한 내용은 [회원(라이선스)](usage/member.md) 문서를 참고하세요.

## 특징

- **지원 증권사**
    - 메리츠증권 (iMERITZ XII)
- **지원 주식장**
    - 미국장 (예: SOXL)
- **지원 자동매매 방법**
    - 구글 시트매매 — 계정별로 서로 다른 매매법(시트)을 연결해 운용
    - 다양한 매매법 프리셋: 터보운전법 · 안전운전법 · 풍차매매법 · 프리미엄법 · 용돈법
- **다중 계정 지원** — 한 번 실행으로 여러 계좌를 순차 매매
- **아침 데이터 결산** — 주문 없이 잔고·시세만 수집해 시트에 기록
- **예약 실행 / 자동 예약** — 미국장 시각(프리장·정규장 시작/종료)에 맞춰 자동 구동
- **텔레그램 알림** — 실행 결과·오류를 텔레그램으로 전송

!!! warning "투자 위험 안내"
    자동매매는 실제 계좌·실제 돈으로 주문을 냅니다. 프로그램 오류, 증권사 화면 변경, PC 환경 차이 등으로 예상과 다르게 동작할 수 있습니다. **처음에는 반드시 소액으로 테스트**하시고, 문제가 없다고 판단되면 금액을 늘리시길 권합니다. 투자 결과에 대한 책임은 사용자 본인에게 있습니다.

## 저작권 및 사용 조건

!!! danger "무단 도용·수정·재배포 금지"
    본 프로그램(FDTS AutoTrading)의 저작권은 **제작자(로이킴에버)**에게 있습니다.

    - 제작자의 동의 없이 프로그램을 **타인에게 재배포**하거나 **판매**할 수 없습니다.
    - 프로그램·구성 파일·키(서비스 계정 등)를 **무단으로 도용·복제·수정**할 수 없습니다.
    - 배포받은 프로그램은 **본인 사용 목적**으로만 이용해 주세요.

    문의나 협의가 필요하면 [문의하기](contact.md)를 통해 연락 주세요.
