# 🔧 일반 설정

메인 화면 사이드바의 **⚙️ 설정** 을 누르면 설정 창이 열립니다. 설정 창은 **일반 설정 · 계정 설정 · 예약 실행 · About** 네 개의 탭으로 구성됩니다.

이 문서는 첫 번째 **일반 설정** 탭을 안내합니다.
계정은 [계정 설정](accounts.md), 예약은 [예약 실행](schedule.md), 새 버전 설치는 [업데이트](update.md)를 참고하세요.

![설정 - 일반 설정 탭](../img/settings_general.png)

## 항목별 상세

### 📂 증권HTS 위치
메리츠 HTS(iMERITZ XII)의 실행파일(`imeritz.exe`) 경로입니다.

- **비워두면** 프로그램이 표준 설치 위치(C·D·E·F 드라이브의 `메리츠증권\iMERITZ XII\Main\imeritz.exe`)를 **자동으로 찾습니다.** 보통은 그대로 두어도 됩니다.
- 직접 지정하려면 **"📂 증권HTS 위치" 라벨(파란 글자)을 클릭**하면 파일 선택 창이 열립니다.

### Google Sheet URL
매매에 사용할 **내 구글 시트 주소**입니다. 전체 URL(`https://docs.google.com/…`) 또는 시트 키(URL 가운데의 긴 문자열)를 넣을 수 있습니다.

!!! tip "칸을 더블클릭하면 시트가 열립니다"
    입력칸을 **더블클릭**하면 그 시트가 브라우저로 바로 열려, 주소가 맞는지 확인할 수 있습니다.

### 📄 신규 구글시트 만들기 (템플릿 복사)
매매법 시트가 아직 없다면 이 버튼을 누르세요. **템플릿 시트**가 브라우저에 열립니다. 구글에서 **파일 → 사본 만들기**로 내 계정에 복사한 뒤, 그 사본의 주소를 위 **Google Sheet URL** 에 넣습니다.

### 시트 공유 계정 (중요)
프로그램이 내 시트를 **읽고 쓰려면**, 여기 표시된 **서비스 계정 이메일**을 내 구글 시트의 공유(편집자)에 추가해야 합니다.

1. 옆의 **[복사]** 버튼으로 이메일 주소를 복사합니다.
2. 내 구글 시트 → **[공유]** → 붙여넣기 → 권한 **'편집자'** → 공유.

!!! danger "이 절차를 빠뜨리면 매매가 안 됩니다"
    시트 공유 계정을 편집자로 추가하지 않으면, 앱이 시트를 읽지 못해 **잔고·시세 수집과 주문이 모두 실패**합니다. 자세한 절차는 [준비하기](../prep.md)를 참고하세요.

### Telegram ID / Token (선택)
실행 결과·오류·새 버전 알림을 **텔레그램**으로 받고 싶을 때 봇 정보를 넣습니다. 비워두면 알림만 오지 않을 뿐, 매매는 정상 동작합니다.

### 권장 / 현재 디스플레이
- **권장 디스플레이**: `3840×2160 · 200% 배율`
- **현재 디스플레이**: 지금 내 화면의 해상도·배율을 **자동 감지**해 보여줍니다.
    - **✅ 권장 설정** — 그대로 사용하세요.
    - **⚠ 권장과 다름** — 동작은 하지만, 좌표가 어긋나 문제가 생기면 권장값으로 맞춰 보세요.

!!! tip "배율은 자동으로 맞춥니다"
    배율(100%·150%·200%)은 직접 고르지 않아도 프로그램이 **현재 화면 배율을 자동 감지**해 클릭 위치를 보정합니다. 자세한 내용은 [권장사양](../requirements.md)을 참고하세요.

### 속도 배수(느린 PC)
모든 대기·타임아웃 시간에 곱해지는 **여유 배수**입니다(기본 `1.0`).

- 느린 PC에서 "창을 못 찾음 / 시간초과" 오류가 나면 **`1.5`~`2.0`** 으로 올려 보세요.
- 안전 범위는 `0.5`~`5.0` 입니다.

### 📂 CSV 저장 경로
잔고·시세를 수집할 때 만드는 **임시 CSV 파일**을 저장할 폴더입니다. 보통 기본값(`csv`)이면 충분하고, **라벨을 클릭**하면 폴더를 바꿀 수 있습니다.

설정을 바꾼 뒤 **[💾 일반 설정 저장]** 을 누릅니다.

!!! note "저장 후 재시작"
