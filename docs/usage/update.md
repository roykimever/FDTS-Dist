# ⬆️ 업데이트 (새 버전 설치)

프로그램은 **앱 안에서 자동으로** 업데이트됩니다. 새 버전이 나오면 아래 방법으로 설치하세요.
(업데이트는 설정 탭이 아니라 **메인 화면 사이드바의 버전 버튼**으로 진행합니다.)

## 사이드바의 버전 버튼

메인 화면 **사이드바 하단(설정 위)** 에 현재 버전이 표시됩니다.

| 표시 | 뜻 | 할 일 |
| --- | --- | --- |
| **v5.9.14 최신** | 최신 버전 사용 중 | 할 것 없음 |
| **v5.9.16 업데이트** (강조색) | 새 버전 있음 | **이 버튼을 클릭** → 업데이트 진행 |

버전 상태는 앱 시작 시와 **30분 주기**로 자동 확인됩니다.

## 상황별 — 이럴 때 이렇게 하세요

- **텔레그램으로 "새 버전" 알림을 받았을 때**
    → 앱을 열고, 사이드바의 **버전 버튼("…업데이트")을 클릭**하면 바로 업데이트가 시작됩니다.
- **앱을 새로 시작했을 때 / 실행 버튼을 눌렀을 때**
    → 새 버전이 있으면 **"새 버전 안내" 팝업**이 뜹니다. **[예]** 를 누르세요.
- **자동 실행(예약 매매) 중일 때**
    → 매매를 멈추지 않도록 **텔레그램으로만** 알려줍니다. 매매가 끝난 뒤 위 방법으로 업데이트하세요.

## 업데이트가 진행되는 과정

1. **[예]** 또는 **버전 버튼 클릭** → 새 코드를 자동으로 내려받아 **무결성 검사(해시 확인)**
2. **"교체하고 재시작"** 확인 창 → **[예]**
3. 앱이 **스스로 종료 → 코드만 교체 → 재시작**

!!! tip "설정은 그대로 유지됩니다"
    자동 업데이트는 **프로그램 코드만** 바꿉니다. 로그인·계좌·시트 주소 등 **설정과 인증 정보는 절대 지워지지 않습니다.** (교체 전 백업본도 남겨 두어, 문제 시 이전 버전으로 되돌릴 수 있습니다.)

!!! warning "매매 중에는 업데이트되지 않습니다"
    매매가 실행 중일 때 업데이트를 누르면 안전하게 거부됩니다. 실행이 끝난 뒤 다시 시도하세요.

## 강제 업데이트 · 사용 중지 (안전장치)

돈이 걸린 프로그램이라, 심각한 문제가 있는 버전을 계속 쓰지 않도록 **안전장치**가 있습니다.

- **강제 업데이트** — 너무 오래된 버전은 매매 시작 직전에 막히고, **업데이트해야 계속 쓸 수 있습니다.**
- **사용 중지(킬스위치)** — 특정 버전에 문제가 발견되면 관리자가 그 버전을 **일시적으로 중지**시킬 수 있습니다. 이때는 안내에 따라 최신 버전으로 올려 주세요.

!!! note "인터넷이 잠깐 안 될 때"
    버전 확인이 네트워크 문제로 실패하면, **그날 매매를 놓치지 않도록 경고만 하고 계속 진행**합니다. (연결이 되면 다시 정상 확인됩니다.)

## 자동이 안 될 때 — 수동 재설치

회사 보안 프로그램 등으로 자동 다운로드가 막히면, [다운로드](../download.md) 페이지에서 최신 버전을 받아 **기존 폴더에 덮어써** 설치할 수 있습니다. 이때도 `key.dat`·`env.json`·`users.json`·`log` 등 **설정/인증 파일은 지우지 마세요.**
